# Titanic Survival Prediction
A machine learning project to predict the survival of passengers on the Titanic Ship using the Random Forest algorithm.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the model: `python model.py`

## Features Used
* Pclass (Ticket class)
* Sex
* Age
* SibSp / Parch (Family relations)
* Fare
* Embarked (Port of embarkation)
